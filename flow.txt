Semplice UI che mostra cosa sta puntando la camera, un bottone di cattura immagine.
Quando catturo immagine, mi mostra l'immagine originale.
L'utente deve dare via all'elaborazione dell'immagine 
Mostrare l'immagine elaborata (immagine con solo contorno della lente)
L'utente deve dare via alla conversione/creazione dall'immagine al file dfx(?)
Finito la conversione salvare il file in una cartella con l'immagine originale e immagine elaborata.

Tutto deve essere salvato nella cartella Documenti->Outile->cartella singola elaborazione.

Nomenclatura delle cartelle deve essere data e ora
Nomenclatura delle immagini e file deve essere:
	- Orinal per immagine originale
	- Outlined per immagine elaborata
	- Data per il file prodotto

1.Creare una finestra che mostra la live della camera
2.Aggiungere un area sotto la finestra per inserire i comandi
3.Aggiungere nell'area un bottone chiamato "Cattura"
	Cattura deve scattare la foto e salvarla 
4.Mostrare la foto scattata al posto della live della camera
5.Al posto del bottone "Cattura" mostrare "Elabora"
	Elabora deve elaborare la foto usando outline.py
6.Aggiungere nell'area un bottone "Elabora" e un bottone "Ripeti"
